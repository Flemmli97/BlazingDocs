## Introduction

Fate/Unlimited Block Works is a mod based around the fate franchise from Type-Moon bringin the concept of the holy grail war to minecraft.

## Getting Started

For **Player** please install
- For 1.18: [Pathouli](https://modrinth.com/mod/patchouli) (Fate Guidebook)
- Otherwise: [Modopedia](https://modrinth.com/mod/modopedia) (Grail War Chronicles)

which will provide you with a guidebook containing all gameplay relevant informations.

For **Developers** head over to [Environment Setup](/dev/Environment%20Setup.md)

If you are a **(Modpack/Datapack) Developer** and want to edit loot tables check out [Grailwar loot table](Grailwar%20loot%20table.md)
