Flan has integration with following mods:

## Permission mods
See [https://github.com/Flemmli97/Flan/wiki/Fabric-Permission-API-Nodes](https://github.com/Flemmli97/Flan/wiki/Fabric-Permission-API-Nodes) for permission nodes:  

* [Fabric Permission API/Luckperms](https://www.curseforge.com/minecraft/mc-mods/luckperms)  
* FTBRanks (All platforms)

## Currency mods

Supported currrency mods to be used to buy/sell claimblocks. Mods that use items as currency are NOT included as you can easily configure item values yourself.

* [Gunpowder Currency (Outdated)](https://www.curseforge.com/minecraft/mc-mods/gunpowder-currency)  
* [Impactor (All platforms)](https://modrinth.com/mod/impactor)
* [Octo Economy (Fabric)](https://github.com/ExcessiveAmountsOfZombies/OctoEconomyApi)
* [Diamond Economy (Fabric)](https://www.curseforge.com/minecraft/mc-mods/diamond-economy)
* [Money and Sign Shops (Forge)](https://www.curseforge.com/minecraft/mc-mods/dicemc-money-mod)

## Claim mods

You cannot claim over the claims of those mod except if you own those claims. Can be disabled in the config.

The reverse is not the case! Those mods would need to also do it from their end

* FTBChunks
* GOML Reserved
* Minecolonies
