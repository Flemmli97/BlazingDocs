All commands are in the form `/flan <command>`

The commands here all require OP

|Command | Description
|----|:----|
reload | Reloads the config ingame
add \<x y z> \<x y z> \<dimension> \<player \| +Admin>| Creates a claim with the given positions, dimension and player (or as an admin claim)
unlockDrops \<players> | Unlocks dropped items for the given players
list \<players> | Lists all claims of the given players
bypass | Switches to admin mode ignoring all claims
readGriefPrevention | Parses data from the GriefPrevention plugin to Flan
setAdminClaim \<true \| false> | Toggles adminclaim for the claim you are standing in
listAdminClaims | Lists all admin claims in the current world
adminDelete [all \<player\>] | Force deletes the current claim or deletes all claims from the specified player
giveClaimBlocks \<amount\> \| (base \<amount>) | Gives a player additional claim blocks or adds to the base amount. Use negative numbers to remove them
